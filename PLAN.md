## Tech Principles (lean + free until revenue)
- Static HTML + Tailwind CDN + Vercel = $0 hosting.
- Formspree free tier for capture. Zapier (connected) for routing.
- AI chat: start rule-based (free). Add Grok or Claude API once revenue covers it.
- localStorage analytics (no third-party tracking, privacy-first).
- Never publish home address or unverified phone. Use virtual address.

## KPIs we track weekly
1. Leads captured (Formspree count)
2. Cost per lead by channel (organic vs free-trial ads)
3. Contractors onboarded + retained
4. MRR
5. Cities live + leads/city

## Rules we don't break
- Privacy-first. No selling user data. TCPA-compliant consent on every form.
- No auto-spam posting (gets accounts banned, burns the brand). Human-in-loop.
- Verify before publishing (phone, address, claims).
- Stick to THIS plan. Change it deliberately, in writing, here.
