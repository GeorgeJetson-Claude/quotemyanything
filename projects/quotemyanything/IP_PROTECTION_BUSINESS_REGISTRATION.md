## 3. Trademark Protection (Federal - Protects Brand Nationally)
- **Search First** (free):
  - USPTO Trademark Search: https://tmsearch.uspto.gov/ (new system; search "Quote My Anything", "QuoteMyAnything", "Free Quotes On Anything", "QMA" + "quotes", live/dead marks in classes 35 (business services), 42 (tech/platform)).
  - Common law: Google, X, state DBs, similar services (Thumbtack, Angi, HomeAdvisor).
  - Current checks (2026): No exact matches found for core marks in relevant classes. "QMA" exists in other fields (e.g., medication aide), but descriptive/weak for quotes platform.
- **File TM** (use attorney recommended):
  - USPTO TEAS (https://www.uspto.gov/trademarks/apply): Standard TEAS ~$250-350 per class (word mark + design if logo).
  - Classes: 35 (advertising; business management; comparison shopping for home services), 42 (SaaS/platform for quotes).
  - Include: Word "QUOTE MY ANYTHING", slogan "FREE QUOTES ON ANYTHING!", logo (wordmark + icon from assets).
  - Specimen: Screenshots of site use in commerce (live site counts).
  - Intent-to-use if not yet "in commerce" everywhere.
  - Timeline: 6-18 months; office actions possible (descriptiveness? "quotes on anything" may be refused as descriptive - argue acquired distinctiveness via use/slogan).
- **State TM**: Optional, file with Texas Secretary of State (~$50-100) for extra state protection.
- **Use Symbols**: ™ for unregistered (now), ® after registration. Add to site, marketing, ads, GitHub, docs: "Free Quotes On Anything!™"
- **Monitor/Enforce**: Use Google Alerts, USPTO watch services, attorney for cease/desist if infringed. For sale, registered TM increases value.

**Risk Note**: "Quote My Anything" is somewhat descriptive; strong use (slogan, logos, marketing) builds secondary meaning. File ASAP before others.
